üì± Complete User Experience Flow

New User Journey:

  1. Landing Page - Hero section, features, user types, stats
  2. Registration - Enter name, email, phone, password
  3. Profile Setup - Step 1 - Choose role: Player, Fan, Club, or Sponsor
  4. Profile Setup - Step 2 - Add role-specific details (position, favorite team, etc.)
  5. Profile Setup - Step 3 - Write bio with character counter and live preview
  6. Welcome Tour - 5 personalized steps explaining key features
  7. Main App - Access all features with bottom navigation

Returning User Journey:

  1. Landing Page - Click "Sign In"
  2. Login - Enter credentials (demo mode accepts any)
  3. Main App - Direct access, skipping setup and onboarding
